Multiple search results + embed

Each result has a channel and post id. You can show a link and/or embed the post.

How it works: One post = one embed. For 20 results you have 20 (channel, postId) pairs.
Option A – embed all at once: Output one script tag per result (same src, different data-telegram-post). Heavy if you have many results.
Option B – on demand: Show result cards; when user clicks "Preview", show the embed (iframe or script tag). Lighter and faster.
